# FreeGIF

一个基于 Electron 的免费 GIF 屏幕录制和编辑工具。

## 功能特性

1. **屏幕录制**
   - 点击按钮隐藏软件界面
   - 鼠标框选任意区域进行录制
   - 实时录制提示和控制

2. **GIF 编辑**
   - 即时预览录制的 GIF
   - 帧级别的时间轴控制
   - 裁剪功能（选择开始帧和结束帧）
   - 调整播放速度（每帧延迟时间）
   - 质量压缩设置

3. **导出功能**
   - 自定义质量压缩
   - 预估文件大小
   - 保存为标准 GIF 格式

## 安装

1. 克隆或下载项目到本地
2. 安装依赖：

```bash
npm install
```

3. 启动应用：

```bash
npm start
```

## 使用方法

### 录制 GIF

1. 启动应用后，点击"开始录制"按钮
2. 主窗口会自动隐藏，屏幕变暗
3. 使用鼠标拖动选择要录制的区域
4. 选择完成后，录制自动开始
5. 点击"停止"按钮结束录制

### 编辑 GIF

录制完成后会自动打开编辑器窗口：

1. **播放控制**：使用播放/暂停按钮预览 GIF
2. **帧导航**：使用上一帧/下一帧按钮精确控制
3. **裁剪**：设置开始帧和结束帧，点击"应用裁剪"
4. **速度调整**：拖动滑块改变每帧延迟时间
5. **质量设置**：调整压缩质量（质量越低，文件越小）

### 导出 GIF

1. 在编辑器中调整满意后，点击"导出 GIF"按钮
2. 选择保存位置和文件名
3. 等待导出完成

## 技术栈

- **Electron**: 跨平台桌面应用框架
- **Node.js**: JavaScript 运行环境
- **Canvas API**: 图像处理和渲染
- **MediaRecorder API**: 屏幕录制
- **gifencoder**: GIF 编码库

## 依赖说明

- `electron`: ^28.0.0 - Electron 框架
- `gifencoder`: ^2.0.1 - GIF 编码器

## 注意事项

1. 首次运行需要授予屏幕录制权限
2. 录制大区域或长时间可能产生较大的内存占用
3. 导出时间取决于 GIF 的帧数和质量设置
4. 建议录制时长控制在 10 秒以内以获得最佳性能

## 快捷键

- **ESC**: 取消区域选择

## 开发

开发模式下启动：
