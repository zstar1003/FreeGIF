# FreeGIF

一个基于 Electron 的免费 GIF 屏幕录制和编辑工具。

## 功能特性

- 📹 **屏幕录制**：选择任意区域进行录制
- 🎬 **实时预览**：录制时实时查看内容
- ✂️ **帧裁剪**：精确裁剪 GIF 的开始和结束帧
- ⚡ **速度调整**：自由调整播放速度（25ms - 200ms）
- 🎨 **质量控制**：调整压缩质量以优化文件大小
- 📁 **导入编辑**：导入现有 GIF 文件进行编辑
- 💾 **轻松导出**：一键导出为 GIF 文件
- 🎯 **统一界面**：录制和编辑在同一窗口完成

## 安装

1. 克隆或下载项目到本地
2. 安装依赖：

```bash
npm install
```

3. 启动应用：

```bash
npm start
```

## 使用方法

### 录制 GIF

1. 启动应用，点击 **"开始录制"** 按钮
2. 使用鼠标拖动选择要录制的区域
3. 选择完成后，自动开始录制（实时预览）
4. 点击 **"停止录制"** 结束录制
5. 自动切换到编辑模式

### 编辑 GIF

录制完成后自动进入编辑模式：

- **播放/暂停**：预览 GIF 效果
- **帧导航**：使用"上一帧"/"下一帧"精确查看
- **裁剪帧**：设置开始帧和结束帧，点击"应用裁剪"
- **调整速度**：拖动滑块改变帧延迟时间（越小越快）
- **设置质量**：调整压缩质量（质量越高，文件越大）

### 导入 GIF

1. 点击 **"导入 GIF"** 按钮
2. 选择本地 GIF 文件
3. 进入编辑模式进行编辑

### 导出 GIF

1. 完成编辑后，点击 **"导出 GIF"** 按钮
2. 选择保存位置和文件名
3. 等待导出完成

## 技术栈

- **Electron** - 跨平台桌面应用框架
- **gif.js** - 纯 JavaScript GIF 编码库
- **MediaRecorder API** - 屏幕录制
- **Canvas API** - 图像处理

## 依赖说明

- `electron`: ^28.0.0 - Electron 框架
- `gif.js`: ^0.2.0 - 纯 JavaScript GIF 编码器
- `@electron/remote`: ^2.1.2 - 远程模块支持

## 使用建议

- **录制时长**：建议控制在 10 秒以内，避免文件过大
- **录制区域**：建议不超过 800x600 像素
- **帧率**：默认 10 FPS，最多录制 100 帧
- **质量设置**：70-80% 可以在文件大小和质量间取得良好平衡

## 注意事项

1. 首次运行可能需要授予屏幕录制权限
2. 录制大区域或长时间可能产生较大的内存占用
3. 导出时间取决于 GIF 的帧数和质量设置

## 快捷操作

- **ESC**：取消区域选择
- **重新录制**：编辑模式下点击"开始录制"重新选择区域
